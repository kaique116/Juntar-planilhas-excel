# Juntar-planilhas-excel
Junte duas planilhas excel com este simples programa!


Passo a passo para instalar todas as bibliotecas necessárias:
  
  Windows:

    Instalar o Python, ativando a caixinha "Add Python to PATH"
    1º Jeito:
      pip install pandas openpyxl
  
    2º Jeito:
      python -m pip install pandas openpyxl

      O Tkinter já vem instalado no Windows!
  
  Linux:
    Instalar o Python e o pip pelo comando "sudo apt-get install python3-pip"

    1º Jeito:
      pip3 install pandas openpyxl
      
      sudo apt install python3-pip python3-tk

      python3 -m pip install --user pandas openpyxl
  
  Mac OS:

    Instalar o Python pelo comando: "brew install python"
    1º Jeito:
      pip3 install pandas
    2º Jeito:
    python3 -m pip install pandas
    O Tkinter já vem instalado no Mac OS!
